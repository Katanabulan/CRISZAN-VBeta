1. Apa itu CSS?

Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan format dari dokumen HTML. CSS mengontrol bagaimana elemen HTML ditampilkan di browser.

Fungsi CSS

  • Mengatur warna teks dan background
  • Mengatur font dan ukuran teks
  • Mengatur layout dan posisi elemen
  • Membuat design responsive
  • Menambahkan animasi dan transisi

Hubungan HTML dan CSS

HTML = Struktur bangunan (dinding, pintu, jendela)
CSS = Desain dan dekorasi (cat, furniture, tata letak)

2. Cara Memakai CSS

1. Inline CSS

CSS ditulis langsung dalam atribut style elemen HTML. 

<h1 style="color: blue; font-size: 24px;">
    Judul Berwarna Biru
</h1>
<p style="background: yellow; padding: 10px;">
    Paragraf dengan background kuning
</p>

2. Internal CSS

CSS ditulis dalam tag <style> di dalam <head>. 

<head>
    <style>
        h1 {
            color: blue;
            font-size: 24px;
        }
        p {
            background: yellow;
            padding: 10px;
        }
    </style>
</head>

3. External CSS

CSS ditulis dalam file terpisah (.css) dan dihubungkan dengan HTML.

File style.css:

h1 {
    color: blue;
    font-size: 24px;
}
p {
    background: yellow;
    padding: 10px;
}

File HTML:

<head>
    <link rel="stylesheet" href="style.css">
</head>

Rekomendasi

Gunakan External CSS untuk project yang serius karena lebih terorganisir dan mudah maintenance.

3. Selector CSS

Apa itu Selector?

Selector digunakan untuk "memilih" elemen HTML yang ingin di-styling.

Jenis-jenis Selector

1. Element Selector

Memilih elemen berdasarkan tag name.

p {
    color: red;
}
h1 {
    font-size: 32px;
}

2. Class Selector

Memilih elemen berdasarkan class. Diawali dengan titik (.)

.important {
    font-weight: bold;
    color: red;
}
.box {
    border: 1px solid black;
    padding: 20px;
}

Penggunaan di HTML:

<p class="important">Teks penting</p>
<div class="box">Kotak</div>

3. ID Selector

Memilih elemen berdasarkan ID. Diawali dengan pagar (#)

#header {
    background: blue;
    color: white;
}
#main-content {
    width: 80%;
    margin: 0 auto;
}

Penggunaan di HTML:

<div id="header">Header Website</div>
<div id="main-content">Konten Utama</div>

Perbedaan Class vs ID

  • Class bisa dipakai banyak elemen
  • ID harus unik, hanya untuk satu elemen
  • Gunakan class untuk styling yang berulang
  • Gunakan ID untuk elemen yang spesifik

4. Warna dan Background

Mengatur Warna Teks

p {
    color: red;               /* Nama warna */
    color: #ff0000;           /* Hex code */
    color: rgb(255, 0, 0);    /* RGB */
    color: rgba(255, 0, 0, 0.5); /* RGB dengan transparansi */
}

Mengatur Background

div {
    background-color: lightblue;    /* Warna background */
    background-image: url('gambar.jpg'); /* Gambar background */
    background-repeat: no-repeat;   /* Tidak diulang */
    background-position: center;    /* Posisi tengah */
    background-size: cover;         /* Cover seluruh area */
}

/* Shorthand */
div {
    background: lightblue url('gambar.jpg') no-repeat center;
}

Contoh Praktis

.warning {
    color: white;
    background-color: red;
    padding: 10px;
    border-radius: 5px;
}

.success {
    color: white;
    background-color: green;
    padding: 10px;
    border-radius: 5px;
}

5. Styling Teks

Font Properties

p {
    font-family: Arial, sans-serif;  /* Jenis font */
    font-size: 16px;                 /* Ukuran font */
    font-weight: bold;               /* Ketebalan */
    font-style: italic;              /* Gaya miring */
    line-height: 1.5;                /* Jarak antar baris */
}

Text Properties

h1 {
    text-align: center;      /* Rata tengah */
    text-decoration: underline; /* Garis bawah */
    text-transform: uppercase; /* Huruf kapital */
    letter-spacing: 2px;     /* Jarak antar huruf */
    word-spacing: 5px;       /* Jarak antar kata */
}

Google Fonts

Cara menggunakan font dari Google Fonts:

1. Tambahkan di HTML:

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

2. Gunakan di CSS:

body {
    font-family: 'Roboto', sans-serif;
}

6. Box Model

Apa itu Box Model?

Setiap elemen HTML adalah sebuah "kotak" yang terdiri dari:

  • Content - Konten sebenarnya (teks, gambar)
  • Padding - Jarak antara content dan border
  • Border - Garis tepi
  • Margin - Jarak dengan elemen lain

Visualisasi Box Model

┌─────────────────────────────┐
│          Margin             │
│  ┌───────────────────────┐  │
│  │       Border          │  │
│  │  ┌─────────────────┐  │  │
│  │  │    Padding      │  │  │
│  │  │  ┌───────────┐  │  │  │
│  │  │  │  Content  │  │  │  │
│  │  │  └───────────┘  │  │  │
│  │  └─────────────────┘  │  │
│  └───────────────────────┘  │
└─────────────────────────────┘

Contoh Penggunaan

.kotak {
    width: 200px;           /* Lebar content */
    height: 100px;          /* Tinggi content */
    padding: 20px;          /* Jarak dalam */
    border: 2px solid black; /* Garis tepi */
    margin: 10px;           /* Jarak luar */
    
    /* Shorthand */
    padding: 10px 20px;     /* atas-bawah kiri-kanan */
    margin: 10px 20px 30px 40px; /* atas kanan bawah kiri */
}

Box-Sizing

Mengatur bagaimana width dan height dihitung:

.kotak {
    box-sizing: border-box; /* width termasuk padding & border */
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    /* Total width tetap 200px */
}

.kotak2 {
    box-sizing: content-box; /* default */
    width: 200px;
    padding: 20px;
    border: 2px solid black;
    /* Total width = 200 + 40 + 4 = 244px */
}

7. Layout Dasar

Display Property

/* Block - mengambil lebar penuh */
div, p, h1 {
    display: block;
}

/* Inline - hanya mengambil space yang dibutuhkan */
span, a, strong {
    display: inline;
}

/* Inline-block - seperti inline tapi bisa atur width/height */
button, img {
    display: inline-block;
}

/* None - menyembunyikan elemen */
.hidden {
    display: none;
}

Float

Untuk membuat elemen mengapung ke kiri atau kanan.

.gambar {
    float: left;
    margin-right: 20px;
}

.sidebar {
    float: right;
    width: 30%;
}

/* Clear float */
.container::after {
    content: "";
    display: table;
    clear: both;
}

Flexbox Dasar

Layout modern untuk mengatur elemen dalam satu dimensi.

.container {
    display: flex;
    justify-content: space-between; /* Sejajar horizontal */
    align-items: center;           /* Sejajar vertikal */
    flex-direction: row;           /* Arah: row/column */
}

.item {
    flex: 1;  /* Membagi space sama rata */
}

8. Position

Jenis-jenis Position

1. Static (Default)

Posisi normal dalam flow dokumen.

div {
    position: static;
}

2. Relative

Posisi relatif terhadap posisi normalnya.

.box {
    position: relative;
    top: 10px;    /* Geser 10px ke bawah */
    left: 20px;   /* Geser 20px ke kanan */
}

3. Absolute

Posisi relatif terhadap parent yang memiliki position relative/absolute.

.parent {
    position: relative;
}

.child {
    position: absolute;
    top: 0;
    right: 0;
}

4. Fixed

Posisi relatif terhadap viewport (tetap di tempat saat scroll).

.navbar {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
}

5. Sticky

Gabungan relative dan fixed.

.header {
    position: sticky;
    top: 0;
}

9. Responsive Design

Viewport Meta Tag

Wajib ada di HTML untuk responsive design.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Menerapkan CSS berbeda berdasarkan ukuran layar.

/* Desktop */
.container {
    width: 1200px;
    margin: 0 auto;
}

/* Tablet */
@media (max-width: 768px) {
    .container {
        width: 100%;
        padding: 0 20px;
    }
}

/* Mobile */
@media (max-width: 480px) {
    .container {
        padding: 0 10px;
    }
    .menu {
        flex-direction: column;
    }
}

Unit Responsive

/* Relative units */
.container {
    width: 80%;           /* Persentase dari parent */
    font-size: 2rem;      /* Relative ke root font size */
    padding: 2em;         /* Relative ke font size elemen */
}

/* Viewport units */
.hero {
    height: 100vh;        /* 100% tinggi viewport */
    width: 100vw;         /* 100% lebar viewport */
}
