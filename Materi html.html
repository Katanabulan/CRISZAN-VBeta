1. Apa itu HTML?

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang menggunakan tag untuk menandai elemen-elemen dalam halaman.

Fungsi HTML

  • Membuat struktur halaman web
  • Menampilkan teks, gambar, video, dan konten lainnya
  • Membuat form untuk input data
  • Membuat link antar halaman
  • Mengorganisir konten dengan heading, paragraf, list, dll

Contoh Sederhana

<!DOCTYPE html>
<html>
<head>
    <title>Halaman Pertamaku</title>
</head>
<body>
    <h1>Selamat Datang!</h1>
    <p>Ini adalah halaman web pertama saya.</p>
</body>
</html>

2. Struktur Dasar HTML

Bagian-bagian Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar yang terdiri dari beberapa bagian penting: 

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten yang ditampilkan -->
</body>
</html>

Penjelasan Setiap Bagian:

  • <!DOCTYPE html> - Deklarasi bahwa ini dokumen HTML5
  • <html> - Elemen root yang membungkus semua konten
  • <head> - Berisi informasi tentang dokumen (tidak ditampilkan)
  • <body> - Berisi konten yang ditampilkan di browser
  • <meta charset="UTF-8"> - Mengatur encoding karakter
  • <title> - Judul yang muncul di tab browser

3. Tag dan Elemen HTML

Apa itu Tag?

Tag adalah penanda yang diapit oleh tanda kurung siku <>. Tag biasanya berpasangan: tag pembuka dan tag penutup.

Struktur Elemen HTML

<nama_tag>Konten di sini</nama_tag>

Contoh Tag Dasar:

<h1>Ini heading level 1</h1>
<p>Ini adalah paragraf.</p>
<strong>Teks tebal</strong>
<em>Teks miring</em>
<br>  <!-- Baris baru (tidak butuh penutup) -->
<hr>  <!-- Garis horizontal -->

Atribut dalam Tag

Atribut memberikan informasi tambahan tentang elemen. Ditulis dalam tag pembuka. 

<tag nama_atribut="nilai">Konten</tag>
<p class="important">Paragraf penting</p>
<img src="gambar.jpg" alt="Deskripsi">

4. Heading dan Paragraf

Heading (h1 - h6)

Heading digunakan untuk judul dan subjudul. Ada 6 level heading dari h1 (terbesar) sampai h6 (terkecil). 

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<h3>Sub Sub Judul</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

Paragraf (p)

Tag <p> digunakan untuk membuat paragraf teks. 

<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua. 
Teks akan otomatis wrap sesuai lebar container.</p>

Format Teks Lainnya

<strong>Teks Tebal (penting)</strong>
<b>Teks Tebal (hanya visual)</b>
<em>Teks Miring (penekanan)</em>
<i>Teks Miring (hanya visual)</i>
<u>Teks Bergaris Bawah</u>
<mark>Teks Disorot</mark>
<small>Teks Kecil</small>
<sub>Subscript</sub>
<sup>Superscript</sup>

5. List (Daftar)

Unordered List (ul)

Daftar tanpa urutan tertentu (bullet points). 

<ul>
    <li>Item pertama</li>
    <li>Item kedua</li>
    <li>Item ketiga</li>
</ul>

Ordered List (ol)

Daftar dengan urutan (nomor/huruf). 

<ol>
    <li>Langkah pertama</li>
    <li>Langkah kedua</li>
    <li>Langkah ketiga</li>
</ol>

Description List (dl)

Daftar dengan istilah dan deskripsi. 

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

6. Link (Hyperlink)

Tag Anchor (a)

Tag <a> digunakan untuk membuat link ke halaman lain atau bagian dalam halaman.

Link ke Halaman Lain

<a href="https://www.google.com">Pergi ke Google</a>
<a href="about.html">Halaman About</a>
<a href="folder/document.pdf">Download PDF</a>

Atribut Penting untuk Link

  • href - URL tujuan
  • target - Cara membuka link
  • title - Tooltip saat hover
<a href="https://example.com" 
   target="_blank" 
   title="Buka di tab baru">
    Klik di sini
</a>

Link ke Bagian dalam Halaman

<!-- Buat anchor -->
<h2 id="bagian1">Bagian 1</h2>

<!-- Link ke anchor -->
<a href="#bagian1">Pergi ke Bagian 1</a>

7. Gambar (img)

Tag Image (img)

Tag <img> digunakan untuk menampilkan gambar. Ini adalah tag self-closing (tidak butuh tag penutup).

Atribut Penting untuk Gambar

  • src - Sumber file gambar
  • alt - Teks alternatif (wajib untuk accessibility)
  • width - Lebar gambar
  • height - Tinggi gambar
  • title - Tooltip saat hover
<img src="gambar.jpg" alt="Deskripsi gambar">
<img src="images/photo.png" alt="Foto" width="200" height="150">
<img src="https://example.com/image.jpg" alt="Gambar dari web">

Best Practices

  • Selalu gunakan atribut alt untuk accessibility
  • Gunakan format gambar yang tepat (JPG untuk foto, PNG untuk grafik)
  • Optimalkan ukuran file untuk loading yang cepat

8. Tabel

Struktur Tabel Dasar

Tabel digunakan untuk menampilkan data dalam format baris dan kolom. 

<table border="1">
    <tr>
        <th>Nama</th>
        <th>Usia</th>
        <th>Kota</th>
    </tr>
    <tr>
        <td>Andi</td>
        <td>25</td>
        <td>Jakarta</td>
    </tr>
    <tr>
        <td>Budi</td>
        <td>30</td>
        <td>Bandung</td>
    </tr>
</table>

Tag-tag dalam Tabel

  • <table> - Container tabel
  • <tr> - Table row (baris)
  • <th> - Table header (sel header)
  • <td> - Table data (sel data)
  • <caption> - Judul tabel

9. Form

Form Dasar

Form digunakan untuk mengumpulkan input dari pengguna.

Kode HTML Form Diatas:

<form action="/submit" method="POST">
    <label for="nama">Nama Lengkap:</label>
    <input type="text" id="nama" name="nama" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>
    
    <label for="pesan">Pesan:</label>
    <textarea id="pesan" name="pesan" rows="4"></textarea>
    
    <button type="submit">Daftar</button>
</form>
