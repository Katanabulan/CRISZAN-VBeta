1. Apa itu PHP?

Pengertian PHP

PHP (PHP: Hypertext Preprocessor) adalah bahasa scripting server-side yang digunakan untuk pengembangan web. PHP dijalankan di server sebelum halaman dikirim ke browser.

Karakteristik PHP

  • Server-side - Dijalankan di server, bukan browser
  • Open Source - Gratis dan komunitas besar
  • Cross-platform - Bisa di Windows, Linux, macOS
  • Database Support - Terintegrasi dengan MySQL, PostgreSQL, dll
  • Easy to Learn - Syntax mirip C dan Java

Contoh Sederhana

<!DOCTYPE html>
<html>
<head>
    <title>Contoh PHP</title>
</head>
<body>
    <?php
        echo "Halo, Dunia!";
    ?>
</body>
</html>

Yang Bisa Dilakukan PHP

  • Membuat halaman web dinamis
  • Mengolah data form
  • Mengirim dan menerima cookies
  • Mengakses database
  • Membuat session user
  • Membuat file (upload, download, edit)

2. Syntax Dasar PHP

Struktur Dasar

Kode PHP ditulis dalam tag <?php ?> dan file disimpan dengan ekstensi .php 

<?php
    // Kode PHP di sini
?>

Komentar dalam PHP

<?php
    // Ini komentar satu baris
    
    # Ini juga komentar satu baris
    
    /*
    Ini adalah
    komentar multi-baris
    */
?>

Output dengan echo dan print

<?php
    // echo - bisa output multiple values
    echo "Hello World!";
    echo "Ini ", "baris ", "kedua";
    
    // print - hanya satu value
    print "Hello World!";
    
    // Output HTML
    echo "<h1>Judul</h1>";
    echo "<p>Paragraf</p>";
?>

Menggabungkan PHP dan HTML

<!DOCTYPE html>
<html>
<head>
    <title>Contoh</title>
</head>
<body>
    <h1><?php echo "Selamat Datang"; ?></h1>
    
    <?php
        $nama = "Budi";
        $umur = 25;
    ?>
    
    <p>Nama: <?php echo $nama; ?></p>
    <p>Umur: <?php echo $umur; ?></p>
</body>
</html>

3. Variabel dan Tipe Data

Membuat Variabel

Variabel PHP diawali dengan tanda $. Nama variabel case-sensitive. 

<?php
    $nama = "Budi";
    $umur = 25;
    $tinggi = 170.5;
    $isActive = true;
?>

Tipe Data PHP

<?php
    // String
    $teks = "Hello World";
    $nama = 'Budi';
    
    // Integer
    $umur = 25;
    $harga = 10000;
    
    // Float/Double
    $berat = 65.5;
    $harga = 99.99;
    
    // Boolean
    $isLogin = true;
    $isAdmin = false;
    
    // Null
    $alamat = null;
?>

Menggabungkan String

<?php
    $nama = "Budi";
    $umur = 25;
    
    // Menggunakan dot (.)
    echo "Nama: " . $nama . ", Umur: " . $umur;
    
    // Dalam double quotes, variabel otomatis diproses
    echo "Nama: $nama, Umur: $umur";
    
    // Single quotes tidak memproses variabel
    echo 'Nama: $nama, Umur: $umur'; // Output literal
?>

Konstanta

Nilai yang tidak bisa diubah.

<?php
    define("NAMA_SITUS", "CRISZAN");
    define("TAHUN_PENDIRIAN", 2024);
    define("PI", 3.14);
    
    echo NAMA_SITUS; // CRISZAN
    echo TAHUN_PENDIRIAN; // 2024
?>

4. Operator

Operator Aritmatika

<?php
    $a = 10;
    $b = 3;
    
    echo $a + $b;  // 13 (Penjumlahan)
    echo $a - $b;  // 7  (Pengurangan)
    echo $a * $b;  // 30 (Perkalian)
    echo $a / $b;  // 3.333 (Pembagian)
    echo $a % $b;  // 1 (Modulus)
    echo $a ** $b; // 1000 (Pangkat)
?>

Operator Perbandingan

<?php
    $x = 5;
    $y = 10;
    
    var_dump($x == $y);   // bool(false)
    var_dump($x != $y);   // bool(true)
    var_dump($x === $y);  // bool(false)
    var_dump($x !== $y);  // bool(true)
    var_dump($x > $y);    // bool(false)
    var_dump($x < $y);    // bool(true)
    var_dump($x >= $y);   // bool(false)
    var_dump($x <= $y);   // bool(true)
?>

Operator Logika

<?php
    $isLogin = true;
    $isAdmin = false;
    
    var_dump($isLogin && $isAdmin); // bool(false) - AND
    var_dump($isLogin || $isAdmin); // bool(true)  - OR
    var_dump(!$isLogin);            // bool(false) - NOT
?>

Operator Penugasan

<?php
    $x = 10;
    $x += 5;  // $x = $x + 5 (15)
    $x -= 3;  // $x = $x - 3 (12)
    $x *= 2;  // $x = $x * 2 (24)
    $x /= 4;  // $x = $x / 4 (6)
    $x %= 4;  // $x = $x % 4 (2)
?>

5. Kondisi (If/Else)

If Statement

<?php
    $umur = 18;
    
    if ($umur >= 18) {
        echo "Anda sudah dewasa";
    }
?>

If...Else

<?php
    $nilai = 75;
    
    if ($nilai >= 70) {
        echo "Anda lulus";
    } else {
        echo "Anda tidak lulus";
    }
?>

Elseif

<?php
    $nilai = 85;
    
    if ($nilai >= 90) {
        echo "Grade: A";
    } elseif ($nilai >= 80) {
        echo "Grade: B";
    } elseif ($nilai >= 70) {
        echo "Grade: C";
    } else {
        echo "Grade: D";
    }
?>

Switch Case

<?php
    $hari = "Senin";
    
    switch ($hari) {
        case "Senin":
            echo "Hari kerja";
            break;
        case "Sabtu":
        case "Minggu":
            echo "Hari libur";
            break;
        default:
            echo "Hari kerja";
    }
?>

Ternary Operator

<?php
    $umur = 20;
    $status = ($umur >= 18) ? "Dewasa" : "Anak-anak";
    echo $status; // Dewasa
?>
