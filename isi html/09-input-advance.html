Input HTML Lanjutan

Tipe input seperti date, tel, number, file, serta atribut required dan pattern.

Contoh Kode

<!DOCTYPE html>
<html><head><meta charset="utf-8"><title>Input Lanjutan</title></head>
<body>
  <form>
    Tanggal: <input type="date" name="tgl"><br><br>
    Telepon: <input type="tel" pattern="[0-9]{10,13}"><br><br>
    Foto: <input type="file" name="foto"><br><br>
    <button type="submit">Daftar</button>
  </form>
</body></html>
