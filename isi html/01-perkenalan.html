Perkenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup untuk membuat struktur halaman web.

Contoh Kode

<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><title>Perkenalan HTML</title></head>
<body>
  <h1>Hello HTML!</h1>
  <p>Ini adalah halaman HTML pertama saya.</p>
</body>
</html>
