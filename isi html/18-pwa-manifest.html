Web Manifest & PWA

File manifest.json dan service worker (JS) diperlukan untuk mengubah situs jadi PWA.

Contoh Kode

<!DOCTYPE html>
<html><head><meta charset="utf-8"><title>PWA</title><link rel="manifest" href="manifest.json"></head>
<body><h1>PWA</h1><p>Tambahkan manifest.json dan service worker.</p></body></html>
